Windows Commands for Docker Deployment
=====================================

Run these commands in Command Prompt or PowerShell:

1. Build Docker Image:
   docker build -t northeastern-chatbot:latest .

2. Tag for Docker Hub:
   docker tag northeastern-chatbot:latest abhisheksagar308/northeastern-chatbot:latest

3. Login to Docker Hub:
   docker login
   (Enter username: abhisheksagar308 and your password)

4. Push to Docker Hub:
   docker push abhisheksagar308/northeastern-chatbot:latest

5. Verify the image:
   docker images

Alternative: Run the batch file
==============================

You can also just double-click on "deploy_windows.bat" to run all commands automatically.
